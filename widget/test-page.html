Lead Form Widget — Test Page

Local testing instructions:
1. Start your local Vercel dev server: npx vercel dev
2. Edit widget/lead-form.html and set WEBHOOK_URL to http://localhost:3000/api/webhook
3. Set WEBHOOK_SECRET to match the value in your local .env
4. Open this file in a browser (or serve with: npx serve widget/)
5. Submit the form below and check the Vercel dev server logs
